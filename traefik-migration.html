NGINX Traefik Migration Tool

Under construction: This is a scaffold for the Traefik → F5 NGINX Ingress Controller migration tool. Content is being added incrementally — see the repository for progress.
Version Reference: This tool is based on Traefik v3.7.1 and F5 NGINX Ingress Controller v5.5.0.

Overview

This interactive guide will help you migrate from Traefik Proxy to the F5 NGINX Ingress Controller (nginx/kubernetes-ingress). Coverage will include:

  • Annotation mappings — Translations for traefik.ingress.kubernetes.io/router.* and traefik.ingress.kubernetes.io/service.* annotations on standard Ingress resources
  • Middleware mappings — Mappings for all 22 OSS HTTP middlewares (StripPrefix, RateLimit, Headers, etc.) to NIC Policy CRDs, annotations, or snippets
  • CRD migration — Worked examples translating Traefik IngressRoute, IngressRouteTCP, and TraefikService to NIC VirtualServer, TransportServer, and Policy CRDs
  • Static-config mappings — Equivalent NIC nginx-config ConfigMap keys for Traefik's common static-config toggles
